# nightshift-014-checksum

Checksum Studio: a small browser tool to generate SHA-1, SHA-256, SHA-384, and SHA-512 hashes for any text input.

## What it does

- Hashes text locally in the browser using Web Crypto API
- Supports multiple algorithms at once
- One-click copy for each hash output
- No backend required

## How to run

```bash
bun install
bun run dev
```

Build for production:

```bash
bun run build
```
